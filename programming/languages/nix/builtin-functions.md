# Builtin Functions

All built-in functions are available through the global [`builtins`](builtin-constants/builtins) constant.

For convenience, some built-ins can be accessed directly:

- [`derivation`](programming/languages/nix/builtin-functions/derivation)
- [`import`](import)
- [`abort`](abort)
- [`throw`](programming/languages/nix/builtin-functions/throw)
