The types are classified as follows:

    - [arithmetic types](fundamental-types/arithmetic-types)
        - [integer types](fundamental-types/arithmetic-types/integer-types)

                - [wide character types](fundamental-types/arithmetic-types/integer-types#^wide-character-types)
            - [signed integer types](fundamental-types/arithmetic-types/integer-types#^signed-integer-types)
                - [standard signed integer types](fundamental-types/arithmetic-types/integer-types#^standard-signed-integer-types)
                - [extended signed integer types](fundamental-types/arithmetic-types/integer-types#^extended-signed-integer-types)
            - [unsigned integer types](fundamental-types/arithmetic-types/integer-types#^unsigned-integer-types)
                - [standard unsigned integer types](fundamental-types/arithmetic-types/integer-types#^standard-unsigend-integer-types)
                - [extended unsigned integer types](fundamental-types/arithmetic-types/integer-types#^extended-unsigned-integer-types)
        - [floating point types](fundamental-types/arithmetic-types/floating-point-types)
- [compound types](compound-types)
    - [reference types](compound-types/reference-types)
    - [pointer types](compound-types/pointer-types)

    - [array types](compound-types/array-types)

    - [class types](compound-types/class-types)
