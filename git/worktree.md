# Git Worktree

A worktree is actual file system directory which contains the repository.

You can use the following commands to manage worktrees.

```
git worktree add
git worktree list
git worktree lock
git worktree move
git worktree prune
git worktree remove
git worktree repair
git worktree unlock
```
