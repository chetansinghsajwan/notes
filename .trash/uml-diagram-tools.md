## LucidChart

> [!info] Intelligent Diagramming | Lucidchart  
> Lucidchart is your solution for visual communication and cross-platform collaboration.  
> [https://www.lucidchart.com/pages/](https://www.lucidchart.com/pages/)  
### Advantages
- Feature rich.
- UML elements available.
- Sync available.
- Android and iOS app available.
- LucidSpark is available for Whiteboard.
### Disadvantages
- Free user can create only 3 boards.
- Free user can use only 60 shapes on each board.
- Desktop app not available.
## Miro

> [!info] The Visual Collaboration Platform for Every Team | Miro  
> Scalable, secure, cross-device, and enterprise-ready team collaboration whiteboard for distributed teams.  
> [https://miro.com/](https://miro.com/)  
### Advantages
- Simple but less featured compared to LucidChart.
- Android and iOS app available.
- Sync available.
- Android app is quite good.
- Desktop app available.
### Disadvantages
- Free user can create only 3 boards.
- Free user can’t use UML elements.
