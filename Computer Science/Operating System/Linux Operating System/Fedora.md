Fedora is a popular open source Linux-based operating system.
Fedora is developed on a six-month to eight-month [release](https://www.techtarget.com/searchsoftwarequality/definition/release) cycle under the Fedora Project. Both the OS and the Fedora Project are financially sponsored and supported by [Red Hat](https://www.techtarget.com/searchdatacenter/definition/Red-Hat).
The Fedora Project releases a new version of Fedora Linux approximately every six months and provides updated packages (maintenance) to these releases for approximately 13 months. This allows users to "skip a release" while still being able to always have a system that is still receiving updates.
Fedora end of life dates are not typically known far in advance with to-the-day accuracy. Fedora has a relatively short life cycle: Release X is supported until one month (4 weeks) after the release of Release X+2 and with approximately 6 months between most versions, meaning a version of Fedora is usually supported for at least 13 months, possibly longer.
