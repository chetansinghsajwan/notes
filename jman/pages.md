# Pages

1. Home Page
	
	This is the home page of our website. User and Company both will start from this page.
	
	It contains signup and login buttons which redirect to their relevant pages. It also contains intro for the website and few recently posted jobs.

---

2. About Page
	
	Contains dummy info about the job-board website and the team behind it.

---

3. Login Page
	
	Same login page for both users and companies.

---

4. Signup Page
	
	Same login page for both users and companies.

---

5. User Home Page
	
	This page is the same as `Home Page` except it's little modified towards the user.

---

6. User Profile Page
	
	Contains info about the user, like image, email, contact info, resume.
	
	It also presents a button to download the resume in `pdf` format.

---

7. Find Job Page
	
	Lists all the job posted by all the companies. Also presents with options to filter and sort the jobs.

---

8. Company Home Page
	
	Contains details about the company like, email, contact email, recently posted jobs, location, owner.
	
	This page also acts as Company Profile Page when user wants to see the details of the company.

---

9. Job Posting Page
	
	A form like page to allow companies to post new jobs.

---

10. Job Info Page
	
	Contains info about the page like, salary, location, skills required.

---

11. Applicants Page
	
	This page lists all applicants that applied for the job.
	
	This page is accessible only to companies through the job info page.
