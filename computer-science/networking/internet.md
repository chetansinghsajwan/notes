# Internet

**brief**:: Global system of connected computer networks.

It is the global system of interconnected [computer-networks](computer-science/networking/computer-networks) that uses the [internet-protocol-suite](computer-science/networking/internet-protocol-suite) to communicate between networks and devices.

It has no single centralized governance in either technological implementation or policies for access and usage; each constituent network sets its own policies.
